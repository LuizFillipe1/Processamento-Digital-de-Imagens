# Processamento-Digital-de-Imagens

Programa realizado na linguam JAVA a qual tem a função de detectar olhos avermelhados causados pelo flash da camêra fotográfica. Após a detectação, é possivel utilizar duas ténicas de binarização sendo elas a de OTSU e a entropia de LiLee. É interessate também testar com diversas imagens a qual há tons avermelados para ver como o programa se comporta.
